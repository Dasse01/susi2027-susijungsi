🎯 왜 이 검색기가 필요한가?
  • 2027 수시·정시 전형 정보는 수십 개 대학의
    수백 개 전형에 분산되어 있어 한눈에 파악 불가
  • 학생부종합·교과·논술·정시를 동시에 비교해야
    최적의 지원 전략 수립이 가능
  • 의치한약수, 과기원, 사관학교 등 특수 전형은
    일반 전형과 완전히 다른 기준 적용
  • 전공자율선택 확대로 모집단위별 유형 비교 필수
  • 수능최저 적용 여부가 실질 합격선에 직결되므로
    빠른 필터링 도구가 반드시 필요
🗂️ 탭별 수록 내용
  • 📋 수시전형 — 교과·종합·논술 전형 200건+, 수능최저·면접 여부
  • 🎓 전공자율선택 — 44개 대학 152개 모집단위 유형 현황
  • 🏥 의치한약수 — 의예·치의예·수의예·약학·한의예 167개 전형
  • 🎓 특수대학 — 과기원 5개교·사관학교·경찰대·전통문화대 27건
  • 📊 정시모집 — 수능 반영비율·가산점·반영지표 68건
🔍 검색 및 필터 사용법
  • 상단 검색창 — 키워드 입력 시 전체 컬럼 즉시 검색
  • 컬럼 ▾ 버튼 — 해당 컬럼값 목록에서 다중 선택/해제
  • 쉼표(,) 다중입력 — 필터 검색창에 "고려대, 연세대" 형태로 입력
  • 🔄 필터 초기화 — 모든 필터와 검색어를 한 번에 초기화
  • 검색결과 건수가 우측 상단에 실시간 표시됨
💡 실전 활용 전략
  • 수능최저 "있음" 필터 → 경쟁 완화 전형 우선 탐색
  • 의치한약수 탭에서 지망 학과 전형 일괄 비교 후
    수능최저 충족 가능 여부 즉시 확인
  • 과기원·사관학교 탭 → 수시 6회 미포함이므로
    반드시 추가 지원 카드로 고려
  • 정시 탭 반영지표 필터(백/혼합/표)로
    유·불리 대학 그룹 분류
  • 전공자율선택 탭 → 입학 후 전공 선택 자유도 비교
